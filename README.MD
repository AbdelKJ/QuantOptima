# 📊 QuantOptima: Multi-Asset Portfolio Optimization & Reporting

QuantOptima is an interactive Streamlit web application designed for professional portfolio construction, optimization, and reporting. It enables users to:

✅ Select asset classes and ETFs  
✅ Download historical price data  
✅ Run multiple portfolio optimization models  
✅ Manually input suggested allocations  
✅ Compare allocations visually and in tabular form  
✅ Generate a professional PDF investment report

---

## 🚀 **Key Features**

- 🏦 **Asset Class & ETF Selection** from uploaded master file
- 📈 **Price Data Download** using Yahoo Finance
- ⚙️ **Portfolio Optimization** with:
  - Equal Weighted
  - Maximum Diversification
  - Minimum CVaR
  - Mean-Variance (Max Sharpe)
- 🛠️ **Manual Allocation Input**
- 🧾 **Side-by-Side Allocation Comparison**
- 📊 **Interactive Pie Charts**
- 📄 **PDF Report Generation** (with tables, charts, headers/footers)
